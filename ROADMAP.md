# Agentic Web Design — Learning Roadmap

> This roadmap will be refined once research documents are reviewed. Current structure is a working draft.

## Goal
Become proficient in designing and building web applications powered by AI agents — covering both the engineering and UX/design dimensions of agentic systems.

---

## Phase 1 — Web Foundations
**Milestone:** Build and deploy a simple interactive web app from scratch.
- HTML, CSS, JavaScript fundamentals
- Responsive design basics
- Intro to a modern framework (React or similar)
- Working with APIs (fetch, async/await)

---

## Phase 2 — AI & LLM Fundamentals
**Milestone:** Make your first API call to an LLM and display the result in a web app.
- How LLMs work (conceptual)
- Prompt design basics
- Claude API / Anthropic SDK
- Streaming responses in the browser

---

## Phase 3 — Tool Use & Agentic Patterns
**Milestone:** Build a web app where an agent uses at least two tools to complete a task.
- Function/tool calling
- Agent loops and reasoning traces
- Handling multi-step tasks
- Error handling and safety

---

## Phase 4 — Agent Frameworks
**Milestone:** Rebuild a prior project using an agent framework.
- Overview of frameworks (LangGraph, CrewAI, AutoGen)
- Multi-agent coordination
- Memory and state management
- RAG (Retrieval-Augmented Generation) basics

---

## Phase 5 — Agentic UX & Interface Design
**Milestone:** Design and build a polished UI for an agentic app with streaming, feedback, and transparency.
- UX patterns for AI interfaces (streaming, loading states, citations)
- Human-in-the-loop design
- Showing agent reasoning to users
- Trust, transparency, and error communication

---

## Phase 6 — Capstone Project
**Milestone:** Ship a complete, publicly accessible agentic web application.
- Full-stack agentic app
- Deployment (Vercel, Fly.io, or similar)
- Evaluation and iteration
